## General Questions

### What is Membercat Studios?

Membercat Studios is a Minecraft content studio dedicated to creating high-quality plugins, mods, modpacks, resource packs, and other Minecraft content. We focus on creating well-documented, polished content that enhances the Minecraft experience for players and server owners.

### What content do you create?

We create a wide variety of Minecraft content including server plugins, client and server mods, curated modpacks, custom resource packs, and developer tools. Our content spans both Java and Bedrock editions, with a focus on quality, performance, and ease of use.

### How do I get started?

Getting started is easy! Browse our documentation to find installation guides for our plugins, mods, and modpacks. Each piece of content includes detailed setup instructions, configuration options, and usage examples to help you get started quickly.

### Do you provide support?

Yes, we provide comprehensive documentation and support for all our content. If you have questions, encounter issues, or need help with installation or configuration, you can reach out through our GitHub repositories or community channels.

### Can I contribute to your projects?

Absolutely! We welcome contributions from the community. Check out our GitHub repositories to see how you can get involved, whether it's reporting bugs, suggesting features, submitting pull requests, or creating resource pack variants.

## Installation Questions

### How do I install a modpack?

Modpacks can be installed using modpack launchers like the Modrinth App, Prism Launcher, or MultiMC. Download the modpack from Modrinth and import it into your launcher.

### How do I install plugins?

Plugins are installed by placing the JAR file in your server's `plugins` folder. Make sure you're using a compatible server software (Paper, Purpur, or Spigot).

### How do I install resource packs?

Resource packs are installed through Minecraft's resource pack menu. Download the pack, open Minecraft, go to Options â†’ Resource Packs, and add the pack.

## Technical Questions

### What Minecraft versions do you support?

Our projects support various Minecraft versions. Check each project's documentation page for specific version requirements.

### Do your plugins work with Paper?

Yes, most of our plugins are compatible with Paper, Purpur, and Spigot servers.

### Can I use multiple modpacks together?

Generally, no. Modpacks are designed as complete packages. However, you can customize them by adding or removing mods if you're familiar with mod management.
