Fractured Expiration is a horror modpack based around the mod "The Broken Script."

## Features

- **[The Broken Script](https://modrinth.com/mod/the-broken-script)** – For the best horror expiranse.
- **Shaders** – For an enhanced creepy experience.
- **Optimization** – Some optimization mods for a better experience.

## Installation

1. Download Fractured Expiration from [Modrinth](https://modrinth.com/modpack/fractured-expiration)
2. Install using your preferred modpack launcher
3. Start building amazing contraptions!

## Requirements

- Minecraft 1.21.1 or later
- NeoForge
- Minimum 6GB RAM allocated
