# CreateCat

A high performance Create modpack to have fun playing with your friends.

CreateCat combines the popular Create mod with performance optimizations, making it perfect for multiplayer servers and creative building projects.

## Features

- **Create Mod**: Full Create mod integration
- **Performance Optimized**: Optimized for multiplayer servers
- **Technology Focus**: Build complex machines and contraptions
- **Multiplayer Ready**: Optimized for playing with friends
- **Fabric**: Built for the Fabric mod loader

## Installation

1. Download CreateCat from [Modrinth](https://modrinth.com/modpack/createcat)
2. Install using your preferred modpack launcher
3. Start building amazing contraptions!

## Requirements

- Minecraft 1.20.1 or later
- Fabric Loader
- Minimum 6GB RAM allocated

## Getting Started

1. Install the modpack and launch Minecraft
2. Read the Create mod documentation for building guides
3. Start experimenting with mechanical contraptions
4. Share your creations with friends!
