Performium Nano is an ultra-lightweight version of Performium, stripped down to the absolute essentials for maximum performance. Perfect for low-end systems or when you need every bit of performance possible.

## Features

- **Barebones**: Only the most essential optimization mods
- **Ultra Lightweight**: Minimal resource usage
- **Fabric**: Built for the Fabric mod loader
- **Maximum Performance**: Optimized for the lowest-end systems

## Installation

1. Download Performium Nano from [Modrinth](https://modrinth.com/modpack/performium-nano)
2. Install using your preferred modpack launcher
3. Launch and experience maximum performance

## Requirements

- Minecraft 1.20.1 or later
- Fabric Loader
- Minimum 2GB RAM allocated

## Differences from Performium

Performium Nano removes several mods from the main Performium pack to achieve even better performance on low-end systems. If you have a powerful computer, we recommend using the full Performium modpack instead.
