# Expanded Adventures

A vanilla+ modpack that enriches Minecraft with new biomes, creatures, and technologies.

Expanded Adventures is a comprehensive modpack that adds new content to Minecraft while maintaining the vanilla feel. Explore new biomes, encounter unique creatures, and discover new technologies.

## Features

- **New Biomes**: Explore diverse new biomes
- **New Creatures**: Encounter unique mobs and animals
- **New Technologies**: Discover new crafting and building options
- **Vanilla+**: Maintains the Minecraft feel you know and love
- **Fabric**: Built for the Fabric mod loader

## Installation

1. Download Expanded Adventures from [Modrinth](https://modrinth.com/modpack/expanded-adventures)
2. Install using your preferred modpack launcher
3. Start a new world and begin your adventure!

## Requirements

- Minecraft 1.20.1 or later
- Fabric Loader
- Minimum 6GB RAM allocated

## Getting Started

1. Create a new world with the Expanded Adventures modpack
2. Explore the new biomes and creatures
3. Experiment with new crafting recipes and technologies
4. Build and create with the expanded content
