# Performium

The only performance modpack you'll ever need.

Performium is a lightweight optimization modpack for Fabric that focuses on improving Minecraft's performance without compromising gameplay. With over 194,000 downloads, it's one of the most popular performance modpacks available.

## Features

- **Lightweight**: Minimal impact on system resources
- **Optimization**: Carefully selected mods for maximum performance gains
- **Fabric**: Built for the Fabric mod loader
- **Regular Updates**: Continuously maintained and updated

## Installation

1. Download Performium from [Modrinth](https://modrinth.com/modpack/performium)
2. Install using your preferred modpack launcher (Modrinth App, Prism Launcher, etc.)
3. Launch and enjoy improved performance!

## Requirements

- Minecraft 1.20.1 or later
- Fabric Loader
- Minimum 4GB RAM allocated

## Performance Tips

- Allocate at least 4GB of RAM for optimal performance
- Close unnecessary background applications
- Use OptiFine-compatible shaders if desired
- Adjust render distance based on your hardware
