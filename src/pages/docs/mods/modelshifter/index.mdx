ModelShifter allows you to change all players into cats, ghasts, and many more custom models completely client-side, which makes it work on every server!

Transform your Minecraft experience by changing player models to various creatures and entities. ModelShifter works entirely client-side, so it's compatible with any server.

## Features

- **Client-Side**: Works on any server without server-side installation
- **Multiple Models**: Transform players into cats, ghasts, and more
- **Custom Models**: Support for custom model configurations
- **Fabric & Quilt**: Compatible with both Fabric and Quilt loaders
- **Easy Configuration**: Simple configuration options

## Installation

1. Download ModelShifter from [Modrinth](https://modrinth.com/mod/modelshifter)
2. Install Fabric or Quilt loader
3. Place the mod in your mods folder
4. Launch Minecraft and configure your preferred models

## Configuration

ModelShifter can be configured through the mod's configuration file or in-game settings. You can set default models for all players or configure specific models per player.

## Supported Models

- Cats
- Ghasts
- And many more custom models!

## Requirements

- Minecraft 1.20.1 or later
- Fabric or Quilt Loader
- Client-side only (no server installation needed)
