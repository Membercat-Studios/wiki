# Getting Started

Welcome to Membercat Studios documentation! This guide will help you get started with our Minecraft content.

## What We Create

Membercat Studios creates high-quality Minecraft content including:

- **Modpacks** - Curated collections of mods for specific gameplay experiences
- **Mods** - Client and server-side modifications
- **Plugins** - Server plugins for Paper, Purpur, and Spigot
- **Resource Packs** - Custom textures, sounds, and visual enhancements

## Quick Start

1. **Browse Our Projects** - Check out our [Modpacks](/docs/modpacks/performium), [Mods](/docs/mods/modelshifter), [Plugins](/docs/plugins/easytpa), and [Resource Packs](/docs/resource-packs/pvpsora)

2. **Choose Your Content** - Select the project that interests you

3. **Follow Installation Guides** - Each project has detailed installation instructions

4. **Enjoy!** - Start using our content and enhance your Minecraft experience

## Finding Documentation

Use the sidebar to navigate between different projects and categories. Each project has its own documentation page with installation instructions, configuration options, and usage guides.

## Need Help?

If you encounter any issues or have questions:

- Check the project's documentation page
- Visit our [GitHub](https://github.com/membercat) repositories
- Check the project's Modrinth page for community support
