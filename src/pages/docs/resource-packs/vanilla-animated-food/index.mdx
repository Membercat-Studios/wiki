Bring your food to life with animated textures! This resource pack uses Minecraft's built-in resource pack animation features to add smooth animations to all food items.

## Features

- **No Mods Required**: Uses vanilla resource pack features
- **Animated Textures**: Smooth animations for all food items
- **Vanilla Compatible**: Works with vanilla Minecraft
- **Lightweight**: Minimal performance impact
- **Regular Updates**: Updated for latest Minecraft versions

## Installation

1. Download Vanilla Animated Food from [Modrinth](https://modrinth.com/resource-pack/vanilla-animated-food)
2. Open Minecraft and go to Options â†’ Resource Packs
3. Click "Open Pack Folder"
4. Copy the downloaded ZIP file into the folder
5. Select Vanilla Animated Food in the resource packs menu
6. Click "Done" and enjoy animated food!

## Requirements

- Minecraft 1.21 (24w45a) or later
- No mods required

## How It Works

This resource pack uses Minecraft's built-in texture animation system introduced in snapshot 24w45a. Food items now have smooth, animated textures that make eating more visually appealing.

## Compatible Items

All food items are animated, including:
- Apples
- Bread
- Cooked meats
- Golden foods
- And more!

## Tips

- Works best with Minecraft 1.21+
- Can be combined with other resource packs
- No performance impact on most systems
