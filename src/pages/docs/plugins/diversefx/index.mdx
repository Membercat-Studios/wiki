# DiverseFX

DiverseFX - a minecraft plugin, which is used to display cool Visuals with the help of Resource Packs!

Add stunning visual effects to your Minecraft server using resource packs. DiverseFX provides an easy way to display particle effects, animations, and visual enhancements.

## Features

- **Visual Effects**: Display cool particle effects and visuals
- **Resource Pack Integration**: Works with custom resource packs
- **Easy Commands**: Simple commands for triggering effects
- **Multi-Server Support**: Works with Bukkit, Paper, Purpur, and Spigot
- **Customizable**: Configure effects and triggers

## Installation

1. Download DiverseFX from [Modrinth](https://modrinth.com/plugin/diversefx)
2. Place the JAR file in your server's `plugins` folder
3. Install the required resource pack
4. Restart your server
5. Configure effects in `plugins/DiverseFX/config.yml`

## Resource Pack

DiverseFX requires a resource pack to display visual effects. Download the resource pack from the plugin's Modrinth page and configure your server to use it.

## Commands

- `/diversefx <effect>` - Trigger a visual effect
- `/diversefx list` - List available effects
- `/diversefx reload` - Reload configuration (admin)

## Permissions

- `diversefx.use` - Use visual effect commands (default: true)
- `diversefx.admin` - Admin commands

## Configuration

Edit `plugins/DiverseFX/config.yml` to:
- Configure available effects
- Set effect parameters
- Customize triggers and conditions

## Requirements

- Bukkit, Paper, Purpur, or Spigot server
- Java 17 or later
- Resource pack (download from Modrinth)
