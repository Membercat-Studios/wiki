A Minecraft plugin that allows Youtube/Twitch donations to trigger events in your Minecraft world using the Streamlabs API!

Connect your Minecraft server with your streaming setup. When viewers donate on YouTube or Twitch, trigger custom events in your Minecraft world.

## Features

- **Streamlabs API**: Full integration with Streamlabs
- **Custom Events**: Create custom in-game events for donations
- **YouTube & Twitch**: Support for both platforms
- **Configurable**: Fully customizable event triggers
- **Multi-Server Support**: Works with Paper and Purpur

## Installation

1. Download Streamlabs Integration from [Modrinth](https://modrinth.com/plugin/streamlabs-integration)
2. Place the JAR file in your server's `plugins` folder
3. Restart your server
4. Configure your Streamlabs API token in `plugins/StreamlabsIntegration/config.yml`

## Setup

1. Get your Streamlabs API token from the Streamlabs dashboard
2. Add the token to the plugin configuration
3. Configure event triggers and actions
4. Test with a donation to verify everything works

## Configuration

Edit `plugins/StreamlabsIntegration/config.yml` to:
- Set your Streamlabs API token
- Configure donation event triggers
- Set up custom commands and actions
- Customize messages and notifications

## Requirements

- Paper or Purpur server
- Java 17 or later
- Streamlabs account with API access
